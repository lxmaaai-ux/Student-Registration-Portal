# 🎓 Student Registration Portal

A modern full-stack Student Registration Portal designed to simplify student registration, authentication, profile management, course management, and administrative operations.

The application features a responsive user interface, secure JWT-based authentication, role-based functionality, and SQL Server database integration.

## ✨ Features

- Student Registration and Login
- Secure Password Hashing
- JWT Authentication
- Protected Routes
- Student Profile Management
- View All Students
- Edit Student Details
- Delete Student Records
- Admin Login
- Admin Dashboard
- Course Management
- Student Statistics
- Responsive User Interface
- Light and Dark Mode
- Modern Animated Backgrounds
- SQL Server Database Integration

## 🔐 Authentication

The application uses JSON Web Tokens (JWT) for secure authentication and authorization.

Passwords are securely hashed using bcrypt before being stored in the database.

## 🚀 Installation and Setup

### 1. Clone the Repository

 git clone https://github.com/lxmaaai-ux/Student-Registration-Portal.git

### 2. Install Backend Dependencies

cd backend

npm install

### 3. Install Frontend Dependencies

cd frontend

npm install

### 4. Configure Environment Variables

Create a `.env` file inside the backend folder and configure the required database credentials and JWT secret.

> Important: Never upload your `.env` file or database credentials to GitHub.

### 5. Start the Backend Server

npm start

### 6. Start the Frontend

npm run dev
